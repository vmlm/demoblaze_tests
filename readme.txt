1. Install python >~3

2. In a terminal, go to the project root and run the correct setup script:
    Windows:  setup.bat
    Linux/MacOS: setup.sh

3. Run behave in the same terminal by writing "behave" and pressing enter.

The script will run all tests and produce the following:
a. Screenshots for the cart with two products, the filled purchase form and the purchase success screen.
b. a log of all the actions taken by the test script on the website, including the ids of the products added.
